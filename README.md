# Amelia Gardens Web

Landing page estática lista para Cloudflare Pages.

## Cloudflare Pages

- Production branch: `main`
- Root directory: `/`
- Build command: dejar vacío
- Build output directory: `.`

Este repositorio debe contener directamente `index.html`, `styles.css`, `script.js` y los demás archivos en la raíz.

## Configuración

Editar `config.js` para actualizar:

- `whatsappNumber`
- `canonicalUrl`
- `formEndpoint`

Actualmente el formulario utiliza WhatsApp como método de envío.
